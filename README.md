* –®–∞–±–ª–æ–Ω (–∫–∞–∫ –∏ –≤—Å—ë –æ–ø–∏—Å–∞–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞) –∏—Å–ø–æ–ª—å–∑—É–µ—Ç XML-–ø–æ–¥–æ–±–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤—Å–µ —Ç—ç–≥–∏ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –∑–∞–∫—Ä—ã—Ç—ã (`<p></p>`) –ª–∏–±–æ –∏–º–µ—Ç—å –∑–∞–∫—Ä—ã–≤–∞—é—â–∏–π —Å–ª—ç—à –≤ –∫–æ–Ω—Ü–µ (`<br />`). –ü—Ä–∏ —ç—Ç–æ–º, –≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç XML, –º–æ–∂–Ω–æ –Ω–µ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å —Å–ø–µ—Ü—Å–∏–º–≤–æ–ª—ã –≤—Ä–æ–¥–µ `<` –∏ `>`, –∞ —Ç–∞–∫–∂–µ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–≤—ã—á–∫–∏ –¥–ª—è –∑–Ω–∞—á–µ–Ω–∏–π –∞—Ç—Ä–∏–±—É—Ç–æ–≤.
* –ö–æ–Ω—Ç—Ä–æ–ª—å–Ω—ã–µ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –¥–ª—è –æ–ø–∏—Å–∞–Ω–∏—è –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏—Ö —á–∞—Å—Ç–µ–π —à–∞–±–ª–æ–Ω–∞ —Ç–∞–∫–∂–µ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è XML-—Ç—ç–≥–∞–º–∏, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, —Å –ø—Ä–µ—Ñ–∏–∫—Å–æ–º `e:`. –î–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è —É–∫–∞–∑—ã–≤–∞—é—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ —Ñ–∏–≥—É—Ä–Ω—ã—Ö —Å–∫–æ–±–æ–∫: `{ #count }`. –î–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –∞—Ç—Ä–∏–±—É—Ç–æ–≤ –ø–∏—à—É—Ç—Å—è –∫–∞–∫ `name={...}`, –æ–¥–Ω–∞–∫–æ –µ—Å–ª–∏ –≤–∞—à —Ä–µ–¥–∞–∫—Ç–æ—Ä/IDE –Ω–µ –ø–æ–Ω–∏–º–∞–µ—Ç —Ç–∞–∫–æ–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –º–æ–∂–Ω–æ –ø–∏—Å–∞—Ç—å `name="{...}"`.
* –ü–æ–≤–µ–¥–µ–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –≤ –≤–∏–¥–µ ES-–º–æ–¥—É–ª—è: –≤—ã —ç–∫—Å–ø–æ—Ä—Ç–∏—Ä—É–µ—Ç–µ –æ–±—ä–µ–∫—Ç—ã –∏ —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ –∏–∑–≤–µ—Å—Ç–Ω—ã Endorphin runtime. –í —ç–∫—Å–ø–æ—Ä—Ç–∏—Ä—É–µ–º—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –ø–µ—Ä–≤—ã–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–º *–≤—Å–µ–≥–¥–∞* (–∫—Ä–æ–º–µ –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å–ª—É—á–∞–µ–≤ —Å –æ–±—Ä–∞–±–æ—Ç—á–∏–∫–∞–º–∏ —Å–æ–±—ã—Ç–∏–π) –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è —ç–∫–∑–µ–º–ø–ª—è—Ä –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ (DOM-—ç–ª–µ–º–µ–Ω—Ç), –∫–æ—Ç–æ—Ä—ã–º –º–æ–∂–Ω–æ —É–ø—Ä–∞–≤–ª—è—Ç—å. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, Endorphin –ø—Ä–æ–¥–≤–∏–≥–∞–µ—Ç —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π –ø–æ–¥—Ö–æ–¥ –∫ –æ–ø–∏—Å–∞–Ω–∏—é –ø–æ–≤–µ–¥–µ–Ω–∏—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –∏ –∏–∑–±–∞–≤–ª—è–µ—Ç –æ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –ø—Ä–æ–±–ª–µ–º —Å `this`.
* –£ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤ –¥–∞–Ω–Ω—ã—Ö: *props* (–≤–Ω–µ—à–Ω–∏–π –∫–æ–Ω—Ç—Ä–∞–∫—Ç, –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –≤ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–Ω–∞—Ä—É–∂–∏), *state* (–≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π –∫–æ–Ω—Ç—Ä–∞–∫—Ç, —É–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è —Å–∞–º–∏–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–º) –∏ *store* (–¥–∞–Ω–Ω—ã–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è). –î–ª—è —É–¥–æ–±—Å—Ç–≤–∞ –≤–Ω—É—Ç—Ä–∏ —à–∞–±–ª–æ–Ω–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π –ø—Ä–µ—Ñ–∏–∫—Å –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ –¥–∞–Ω–Ω—ã—Ö:
    * –î–ª—è –æ–±—Ä–∞—â–µ–Ω–∏—è –∫ –∑–Ω–∞—á–µ–Ω–∏—é –∏–∑ *props* –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å `propName`, —Ç–æ –µ—Å—Ç—å –æ–±—Ä–∞—â–∞—Ç—å—Å—è –∫–∞–∫ –∫ –≥–ª–æ–±–∞–ª—å–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π.
    * –î–ª—è –æ–±—Ä–∞—â–µ–Ω–∏—è –∫ *state* –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ø—Ä–µ—Ñ–∏–∫—Å `#` (–ø–æ –∞–Ω–∞–ª–æ–≥–∏–∏ —Å –ø—Ä–∏–≤–∞—Ç–Ω—ã–º–∏ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏ –∫–ª–∞—Å—Å–æ–≤ –≤ JS): `#stateName`.
    * –î–ª—è –æ–±—Ä–∞—â–µ–Ω–∏—è –∫ *store* –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ø—Ä–µ—Ñ–∏–∫—Å `$`: `$storeName`.

## –®–∞–±–ª–æ–Ω

–ö–∞–∫ –±—ã–ª–æ –æ—Ç–º–µ—á–µ–Ω–æ –≤—ã—à–µ, —à–∞–±–ª–æ–Ω –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –∏–∑ —Å–µ–±—è XML-–ø–æ–¥–æ–±–Ω—ã–π —Å–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º–∏ —Ç—ç–≥–∞–º–∏, –∫–æ—Ç–æ—Ä—ã–µ —É–ø—Ä–∞–≤–ª—è—é—Ç —Å–æ–¥–µ—Ä–∂–∏–º—ã–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞.

### –≠–ª–µ–º–µ–Ω—Ç

–≠–ª–µ–º–µ–Ω—Ç—ã –≤ —à–∞–±–ª–æ–Ω–µ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è —Ç–∞–∫ –∂–µ, –∫–∞–∫ –∏ –≤ HTML: —Å –ø–æ–º–æ—â—å—é —Ç—ç–≥–æ–≤ –∏ –∞—Ç—Ä–∏–±—É—Ç–æ–≤. –¢—ç–≥–∏ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –∑–∞–∫—Ä—ã—Ç—ã –ª–∏–±–æ —Å –ø–æ–º–æ—â—å—é –∑–∞–∫—Ä—ã–≤–∞—é—â–µ–≥–æ —Ç—ç–≥–∞, –ª–∏–±–æ —Å –ø–æ–º–æ—â—å—é –∑–∞–∫—Ä—ã–≤–∞—é—â–µ–≥–æ —Å–ª—ç—à–∞:

```html
<h1>My header</h1>
<p class="intro">First<br />paragraph</p>
```

–ê—Ç—Ä–∏–±—É—Ç—ã –º–æ–≥—É—Ç –∏–º–µ—Ç—å —Å–ª–µ–¥—É—é—â–∏–π –≤–∏–¥:

* `name="value"` ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –∫–∞–≤—ã—á–∫–∞—Ö, –º–æ–∂–Ω–æ —É–∫–∞–∑—ã–≤–∞—Ç—å –ª–∏–±–æ –æ—Ä–¥–∏–Ω–∞—Ä–Ω—ã–µ, –ª–∏–±–æ –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏;
* `name=value` ‚Äî –∫–æ—Ä–æ—Ç–∫–∞—è –∑–∞–ø–∏—Å—å –∑–Ω–∞—á–µ–Ω–∏–π, —Å–æ—Å—Ç–æ—è—â–∏—Ö –∏–∑ –æ–¥–Ω–æ–≥–æ —Å–ª–æ–≤–∞, –∫–∞–≤—ã—á–∫–∏ –º–æ–∂–Ω–æ –Ω–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å;
* `name={expr}` –∏–ª–∏ `name="{expr}" ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ–º –∞—Ç—Ä–∏–±—É—Ç–∞ —è–≤–ª—è–µ—Ç—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ä–∞–±–æ—Ç—ã –≤—ã—Ä–∞–∂–µ–Ω–∏—è `expr`.
* `name="foo {expr} bar"` ‚Äì –∏–Ω—Ç–µ—Ä–ø–æ–ª—è—Ü–∏—è —Å—Ç—Ä–æ–∫ –∏ –≤—ã—Ä–∞–∂–µ–Ω–∏–π: –≤ —É–∫–∞–∑–∞–Ω–Ω–æ–π —Å—Ç—Ä–æ–∫–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ —Ñ–∏–≥—É—Ä–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö –∑–∞–º–µ–Ω—è—é—Ç—Å—è –Ω–∞ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤—ã—Ä–∞–∂–µ–Ω–∏—è. –ê–Ω–∞–ª–æ–≥–∏—á–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç –º–æ–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å —Å –ø–æ–º–æ—â—å—é Template Strings: `name={\`foo ${expr} bar\`}`.
* `name` ‚Äî –±—É–ª–µ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞, –∞–Ω–∞–ª–æ–≥ `name={true}`.
* `{name}` ‚Äî —Å–æ–∫—Ä–∞—â—ë–Ω–Ω–∞—è –∑–∞–ø–∏—Å—å –≤–∏–¥–∞ `name={name}`, —Ç–∞–∫–∞—è –∂–µ –∑–∞–ø–∏—Å—å –¥–æ—Å—Ç—É–ø–Ω–∞ –∏ –¥–ª—è state –∏ store: `{#enabled}` ‚Üí `enabled={#enabled}`, `{$data}` ‚Üí `data={$data}`.

Endorphin —Ä–∞–∑–ª–∏—á–∞–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ —Ç—ç–≥–æ–≤: –æ–±—ã—á–Ω—ã–µ HTML-—ç–ª–µ–º–µ–Ω—Ç—ã –∏ DOM-–∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã. –ü–æ—Å–ª–µ–¥–Ω–∏–µ –∏–º–µ—é—Ç –¥–µ—Ñ–∏—Å –≤ –Ω–∞–∑–≤–∞–Ω–∏–∏ (–ø–æ –∞–Ω–∞–ª–æ–≥–∏–∏ —Å –≤–µ–±-–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞–º–∏):

```html
<h1>My header</h1>
<my-component enabled data={items} />
```

–£ –æ–±—ã—á–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏ DOM-–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ –Ω–µ–º–Ω–æ–≥–æ –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –ø–æ–≤–µ–¥–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–æ–≤:

* –î–ª—è –æ–±—ã—á–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∞—Ç—Ä–∏–±—É—Ç—ã –≤—ã–≤–æ–¥—è—Ç—Å—è –∫–∞–∫ –æ–±—ã—á–Ω—ã–µ HTML-–∞—Ç—Ä–∏–±—É—Ç—ã, –Ω–æ –¥–ª—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ –∞—Ç—Ä–∏–±—É—Ç—ã ‚Äî —ç—Ç–æ *props*. –î–ª—è —É–¥–æ–±—Å—Ç–≤–∞ —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏ props —Ç–∞–∫–∂–µ –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –∫–∞–∫ HTML-–∞—Ç—Ä–∏–±—É—Ç—ã —É —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞. –ê –¥–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å —Å–µ–º–∞–Ω—Ç–∏–∫–µ HTML, –Ω–∞–∑–≤–∞–Ω–∏—è –∞—Ç—Ä–∏–±—É—Ç–æ–≤ –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É—é—Ç—Å—è –∏–∑ *camelCase* –≤ *kebab-case*: `<my-component someItems={items}>` –≤ DevTools –æ—Ç–æ–±—Ä–∞–∑–∏—Ç—Å—è –∫–∞–∫ `<my-component some-items="{}">`. HTML-–∞—Ç—Ä–∏–±—É—Ç—ã —É DOM-–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ –Ω–æ—Å—è—Ç —á–∏—Å—Ç–æ –∏–Ω—Ñ–æ—Ä–º–∞—Ç–∏–≤–Ω—ã–π —Ö–∞—Ä–∞–∫—Ç–µ—Ä –∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ CSS-—Å—Ç–∏–ª–∏–∑–∞—Ü–∏–∏ –∏ –æ—Ç–ª–∞–¥–∫–∏ –∫–æ–¥–∞, –∞ —Å–∞–º–∏ –¥–∞–Ω–Ω—ã–µ –¥–æ—Å—Ç—É–ø–Ω—ã –≤ —Å–≤–æ–π—Å—Ç–≤–µ `.props` —ç–ª–µ–º–µ–Ω—Ç–∞. –£ –æ–±—ã—á–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∞—Ç—Ä–∏–±—É—Ç—ã —è–≤–ª—è—é—Ç—Å—è –∏—Å—Ç–æ—á–Ω–∏–∫–æ–º –¥–∞–Ω–Ω—ã—Ö, —Ç–æ –µ—Å—Ç—å –≤–ª–∏—è—é—Ç –Ω–∞ —Ä–∞–±–æ—Ç—É —ç–ª–µ–º–µ–Ω—Ç–∞.
* –î–ª—è –æ–±–æ–∏—Ö —Ç–∏–ø–æ–≤ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ø—Ä–∏–≤–µ–¥–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π –∞—Ç—Ä–∏–±—É—Ç–æ–≤ –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –≤ HTML:
    * —Å—Ç—Ä–æ–∫–∏ –∏ —á–∏—Å–ª–∞ –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –∫–∞–∫ –µ—Å—Ç—å;
    * —Ñ—É–Ω–∫—Ü–∏—è –æ—Ç–æ–±—Ä–∞–∂–∞–µ—Ç—Å—è –∫–∞–∫ `ùëì`;
    * –º–∞—Å—Å–∏–≤ –æ—Ç–æ–±—Ä–∞–∂–∞–µ—Ç—Å—è –∫–∞–∫ `[]`;
    * –ø—Ä–æ—á–∏–µ –Ω–µ–ø—É—Å—Ç—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –∫–∞–∫ `{}`;
    * –¥–ª—è –±—É–ª–µ–≤—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π `true` –≤—ã–≤–æ–¥–∏—Ç –∞—Ç—Ä–∏–±—É—Ç —Å –ø—É—Å—Ç—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º (`<input disabled={true} />` ‚Üí `<input disabled />`), `false` –Ω–µ –≤—ã–≤–æ–¥–∏—Ç –∞—Ç—Ä–∏–±—É—Ç —Å–æ–≤—Å–µ–º (`<input disabled={false} />` ‚Üí `<input />`).
